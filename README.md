# AI Personalized Workout & Diet Planner

An intelligent web-based fitness planning system that generates personalized workout and meal recommendations based on user body metrics and fitness goals.

## Features

- BMI Calculation
- Daily Calorie Target Estimation
- Goal-based Workout Recommendation
- Diet-based Meal Suggestion
- Interactive Dashboard UI
- Cloud Deployment via Streamlit

## Tech Stack

- Python
- Streamlit
- Pandas
- Scikit-learn
- Git & GitHub

## How It Works

1. User inputs age, gender, weight, height, goal, and experience level.
2. System calculates BMI and BMR.
3. Filters workout dataset based on goal and experience.
4. Filters meal dataset based on calorie target and diet type.
5. Displays personalized plan.
